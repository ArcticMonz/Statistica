Basic notions in Statistics

Population

Population is the entire set of items from which data is drawn for a statistical study. It can be a group of individuals, a set of items etc. It constitutes the data pool for a study. The items of this set are denominated Statistical Units.

Statistical Unit

A statistical unit is the basic entity or element about which data is collected in a statistical survey or study. It represents the individual entity that is observed or measured, such as an individual or an item. The phenomenons that we can observe on each statistical unit are called characters or variables. A character is more general and abstract, while a variable is more precise. For example, if we consider a person as a statistical unit, we can take as a variable the weight of the person and define it in a quantitative way (for example 75 kg) or a qualitative way (for example low, medium, high).

Distribution

In Statistics, Distribution refers to how the values of a variable or dataset are spread out or arranged. It describes the frequency with which different values occur and can be represented in various ways, such as tables, charts, or mathematical functions. Types of Distribution

  • Frequency Distribution: Shows how often each value or range of values appears in a dataset. This can be represented using a histogram for quantitative data or a bar chart for qualitative data.
  • Probability Distribution: Defines the probability that a random variable will take on different values. Probability distributions can be discrete (like the roll of a die) or continuous (such as the normal distribution for quantitative data).
  • Normal Distribution: Also known as the “Gaussian distribution,” it is a symmetrical, bell-shaped distribution where values are concentrated around the mean. Many natural phenomena tend to follow a normal distribution.
Understanding the distribution of data is essential in statistical analysis because it provides insights into central tendency (such as mean and median), variability (such as variance and standard deviation), and shape (such as skewness and kurtosis). Knowing the distribution helps in making predictions, making decisions, and interpreting statistical results effectively.
